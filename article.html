Accessibility

Compared to years ago, our lives today are super easy. One of the factors that makes our everyday life easy like this is the Web. Through the web, we can talk to tons of friends at the same time no matter how distant they are from us, we can order food, rent a hotel room, plan a trip, watch videos and our favorite shows, check our bank status and even pay bills, amazing right? Not for everybody. According to the world bank, 15% of the global population experiences some form of disability. This means that for one billion people, their life is not as easy as it should be. So how we as web developers can help them to have the commodity of using the web just like the rest of the world? The answer is simple. It is called accessibility.

Fortunately, today we already have a Web Accessibility Standards created by W3C, that help us as web developers to make websites accessible to everybody. According to them, the disabilities that affects the access to the web are auditory, cognitive, neurological, physical, speech and visual. For each one of those, there is a way to make the user life easier. W3C lists 3 different types of components that put the web together, and for each one of those, there is a standard for web developers to follow to make the web accessible, and here they are.

First is the Web content, it refers to any part of the website, including, images, forms, multimedia, texts and etc. Here is the link to the Web Content Accessibility Guidelines.

The second one is the User agents, it is the software that people use to access the web, like desktop browsers, voice browsers, mobile phones, assistive technology and etc. Here is the link to the User Agent Accessibility Guidelines.

The third one is the Authoring tools, it is the software developers use to create all the web content, like code editors, databases and other tools. Here is the link to the Authoring Tool Accessibility Guidelines.

Social Media Optimization / SMO

The first question is: what is social media optimization? Also known as SMO, social media optimization is basically the optimization of brands throughout social media. In other words, is making sure companies' websites and social media profiles are optimized for social media users through its platform. Have you ever opened a third party link through Facebook and the webpage just did not work or look good through your mobile display? So, that page was not optimized for social media.

Social media has a massive number of users and potential clients. Social media is part of our daily lives, Instagram, facebook, twitter, Snapchat, and many more platforms are names that we know and recognize even just by the logo. To show how massive Social media is today, a poll from Pew Research Center shows that 86% of U.S. Adults Aged between 18 and 29 are social media users. Moreover, 69% of them use it at least once a day. So, as web developers, the awareness of optimizing websites for social media is a must.

As web developers, there are some best practices to help improve the social media optimization of websites.

The first one is making the loading time of the website faster. Hostway shows steps to improve the loading time of websites. They cover some amazing points like Minify JavaScript and CSS, enable HTTP keep-alive response headers, combining images into CSS sprites and many more. Here is a link to their article.

The second one is making sure the website has easy navigation. According to Lyfe Marketing, "The less time it takes for people to find what they’re looking for, the better." It is all about the user experience. Here is an article from butterfly.com that shows 10 steps to improve websites' navigation.
